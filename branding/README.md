# Branding — Phoenix Digital Omnichannel

## Fonte

`source/phoenix_digital_agents_omnichannel.png`

## Regenerar assets

```bash
python3 scripts/phoenix-generate-branding.py
```

Isso atualiza `branding/` e copia para `chatwoot/public/brand-assets/` e favicons.

## Estrutura

- `logos/` — login, sidebar, header, thumbnail  
- `favicons/` — 16, 32, 96, 512  
- `social-previews/` — OG 1200×630  
- `email-assets/` — logo para templates de email
