# Regras de Agentes

- Nunca aumentar orcamento sem aprovacao explicita.
- Nunca publicar campanha sem produto valido em estoque.
- Nunca excluir campanhas, apenas pausar.
- Sempre registrar justificativa de cada decisao do agente.
- Toda acao de risco medio/alto gera `ApprovalRequest`.
- Toda avaliacao de criativo passa por `MetaComplianceAgent`.
